# 別紙3 職員体制

(霞台市公共施設予約管理システム構築及び運用保守業務 提案依頼書 別紙)

## 1. 発注者側プロジェクト体制(構築期間中)

| 役割 | 所属・人数 | 関与度 |
|---|---|---|
| プロジェクト責任者 | 総務部長(1名) | 工程判定・検収承認 |
| プロジェクトマネージャ | 情報政策課長(1名) | 全体統括 |
| 事務局 | 情報政策課 係長1名、主事1名 | 専任ではない(他業務と兼務、本業務への充当は概ね5割) |
| 業務担当 | スポーツ振興課2名、生涯学習課2名、文化振興課1名 | 要件確認・受入テスト(兼務) |
| 会計・収納担当 | 会計課1名 | 収納・財務会計連携の確認(兼務) |

- 情報政策課の体制は課長以下5名であり、本業務のほか基幹業務システムの標準化対応、庁内ネットワーク運用等を所管している。
- 本市にはアプリケーション開発の専門職員はいない。職員の IT リテラシーは一般的な事務職員水準であり、特別な技術知識を前提としないこと。

## 2. システム利用職員(稼働後)

| 区分 | 人数 | 主な利用内容 |
|---|---|---|
| システム管理者(情報政策課) | 2名 | アカウント管理、マスタ承認、ログ確認 |
| 施設所管課職員 | 約15名 | 施設マスタ保守、減免承認、統計帳票 |
| 各施設窓口職員(会計年度任用職員を含む) | 約40名 | 窓口受付、代行予約、現金収納 |
| 指定管理者職員 | 約20名 | 指定管理施設(体育館・文化会館等8施設)の窓口受付・施設管理 |

- 窓口職員は人事異動・年度更新による入替りが多く、年度当初に毎年新任者が発生する前提で操作習得性を考慮すること(NFR-C06 参照)。

|---|---|---|

| マスタ保守(料金・コマ等) | ○(施設所管課) | 操作支援 |
