# 別紙1 現行システム概要

(霞台市公共施設予約管理システム構築及び運用保守業務 提案依頼書 別紙)

## 1. 現行システムの概要

| 項目 | 内容 |
|---|---|
| システム名称 | 霞台市施設予約システム(現行) |
| 導入時期 | 平成24年10月(以後、平成29年度にハードウェア更改1回) |
| 提供形態 | パッケージ製品のオンプレミス導入(庁舎内サーバ室設置) |
| 開発・保守事業者 | 株式会社A社(現行パッケージの販売・保守を令和9年度末で終息予定と表明) |
| サーバ構成 | Web/AP サーバ1台、DB サーバ1台、バックアップ装置1台(いずれも物理サーバ) |
| OS / DB | Windows Server 2016 / SQL Server 2016 |
| 利用者向け機能 | 空き照会、先着予約、抽選申込(体育施設のみ)、予約照会・取消 |
| 職員向け機能 | 窓口受付、施設マスタ保守、料金算定、統計帳票 |
| 決済 | 非対応(窓口現金収納のみ。収納情報は手作業で財務会計へ転記) |

## 2. 現行の業務運用

1. 施設予約の受付チャネルは、現行システムのオンライン予約、各施設窓口、電話の3系統である。電話受付分は窓口職員が現行システムに代行入力している。
2. 公民館12館のうち4館は現行システム未対応であり、紙台帳で運用している。新システムでは全34施設を対象とする。
3. 抽選は体育施設(体育館、テニスコート、グラウンド、野球場)のみ毎月1日〜7日に翌々月分の申込を受け付け、8日に抽選を実施している。抽選申込期間の初日にアクセスが集中する傾向がある。
4. 利用料金の減免(市内団体の公益活動等)は紙申請で受け付け、施設所管課が決裁の上、窓口で減免後料金を収納している。

## 3. 利用者

本システムの利用者は、住民(施設利用者)及び市職員(施設所管課職員、各施設窓口職員、指定管理者職員を含む)である。

- 住民は自宅等の PC・スマートフォンから利用する。
- 市職員及び指定管理者職員は、執務室及び各施設窓口の事務端末から利用する。

## 4. 現行システムの課題(再掲・補足)

1. ハードウェア保守期限(令和9年度末)及びパッケージ保守終息により、現行構成の継続は不可能である。
2. サーバ室の空調・電源・ラックの維持費用が年額約1,200千円発生している。
3. スマートフォン向け画面が未対応(PC 画面の縮小表示)であり、若年層・子育て世代からの改善要望が多い。
4. 抽選申込集中時に画面遷移が極端に遅くなる事象が毎月発生しており、苦情につながっている。
